# mental-health-app
Prototype for mental health web-app using Patient Health Questionnaire (PHQ-9)

## Prerequisites
- node must be installed

## Install

```
cd mental-health-app
npm install
npm start

```

Go to http://localhost:3000
- click on Questionnaire
